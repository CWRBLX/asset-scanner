# asset-scanner
Roblox image asset scanner using Tensorflow and Node.js


## Install steps
1. Clone repo
2. `npm i` to install packages
3. Replace `example.proxies.txt` with `proxies.txt`, and fill with your own proxies. This code uses Webshare proxies, which come in the format `IP:PORT:USERNAME:PASSWORD`
4. Run the code with `node app.js`
